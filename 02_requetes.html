Cours SQL - Requêtes - # 2

INSERT, UPDATE et DELETE

1- INSERT INTO, requête d'insertion

  • 1ère méthode : INSERT INTO employes (prenom, nom, sexe, service, date_embauche, salaire) VALUES ('Alexis', 'Richard', 'm', 'informatique', '2011-12-28', 1800);
  • Afficher cet enregistrement on le cherche par le nom de famille : SELECT * FROM employes WHERE nom = 'Richard'; ou bien on cherche le dernier enregistrement SELECT * FROM employes ORDER BY id_employes DESC LIMIT 1;
  • 2ème méthode : INSERT INTO employes VALUES (NULL, 'Jeanne', 'Richy', 'f', 'informatique', '2021-01-28', 1800);
  • NULL nous permet de laisser le champ id_employes se remplir seul car il est auto-incrémenté

2- UPDATE, requête de mise à jour

  • Augmenter le salaire de l'employée, Jeanne, dont l'id est 992UPDATE employes SET salaire = 1900 WHERE id_employes = 992;
  • On utilise toujours des ID pour mettre à jour les informations de la BDD car c'est le seul élément unique
  • Pour faire plusieurs mise à jour en même temps il faudra respecter l'ordre des colonnes et les séparer par une virgule : UPDATE employes SET service = 'commercial', salaire = 2000 WHERE id_employes = 992;

3- DELETE, requête de suppression

  • Supprimer un enregistrement avec DELETE : DELETE FROM employes WHERE id_employes = 388;
  • Supprimer plusieurs enregistrements, ici on supprime toutes les personnes du service informatique : DELETE FROM employes WHERE service = 'informatique';

4- Exercice

  1. Afficher la profession de l'employé 547 :SELECT nom, prenom, service FROM employes WHERE id_employes = 547;
  2. Afficher la date d'embauche d'Amandine :SELECT nom, prenom, date_embauche FROM employes WHERE prenom = 'Amandine';
  3. Afficher le nombre de commerciaux :SELECT COUNT(*) AS 'nbr_commerciaux' FROM employes WHERE service = 'commercial';
  4. Afficher le coût des commerciaux sur une année :SELECT SUM( salaire * 12) AS 'salaire_annuel_commerciaux' FROM employes WHERE service = 'commercial';
  5. Afficher le salaire moyen par service et trier du plus petit au plus grand :SELECT AVG(salaire) AS 'salaire_moyen_par_service', service FROM employes GROUP BY service ORDER BY salaire_moyen_par_service ASC;
  6. Afficher le nombre de recrutement sur l'année 2010 :SELECT COUNT(*) AS 'nbre de recrutements en 2010' FROM employes WHERE date_embauche LIKE '2010%'; ou bien SELECT COUNT(*) AS 'nbre de recrutements en 2010' FROM employes WHERE date_embauche BETWEEN '2010-01-01' AND '2010-12-31'; ou bien SELECT COUNT(*) AS 'nbre de recrutements en 2010' FROM employes WHERE date_embauche >= '2010-01-01' AND date_embauche <= '2010-12-31';
  7. Augmenter le salaire pour chaque employé de 100 € :UPDATE employes SET salaire = salaire + 100;
  8. Afficher le nombre de service :SELECT COUNT(DISTINCT service) AS 'nombre de service' FROM employes;
  9. Afficher les informations de l'employé du service commercial gagnant le salaire le plus élevé :SELECT nom, prenom, service, salaire FROM employes WHERE service = 'commercial' ORDER BY salaire DESC LIMIT 0,1; ou bien
  10. Afficher l'employé ayant été embauché en dernier SELECT * FROM employes ORDER BY date_embauche DESC LIMIT 1; ou bien : SELECT * FROM employes WHERE date_embauche =(SELECT MAX(date_embauche) FROM employes);
