Les fonctions prédéfinies

1- Quelques fonctions

Fonctions prédéfinies
Fonctions prédéfinies Description
SELECT CURDATE() Affiche la date courante : quand on aura besoin de la date du jour dans la reqête
SELECT CURDATE() +0 Affiche la date courante sans tirets, pratique pour nommer un fichier
SELECT CURTIME() Affiche l'heure
SELECT NOW() Date et heure
SELECT DATE_ADD('2021-12-06', INTERVAL 31 DAY) Ajoute XX jour à la date de départ
SELECT DATE_FORMAT(NOW(),'%d/%m/%Y - %H:%i:%s') autre exemple : SELECT date_rendu, DATE_FORMAT(date_rendu, '%d/%m/%Y') FROM emprunt Change le format de la date du jour au format FR
SELECT CONCAT_WS('-', 'a','b','c') WS whith separator utile pour concaténer un code postal avec une ville
SELECT TRIM(' coucou ') Supprime les espaces en début et en fin de chaîne
